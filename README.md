Reina Azul

This website is for and about Reina Azul for Her and her brand/community she is into digital content, spoken words, and performance arts. It also acts as a direct bridge to Her community (the Blue Family) by letting people instantly connect with Her through custom Whatsaap links.


The  Features

-  It has responsive layout that adjusts to different screen sizes
-  It consists of multiple pages such as the home page, the community page, and a contact page
- It has a cleaned up the header completely by replacing old text placeholders with a high-visibility, responsive interlocking logo asset



I used HTML and CSS with a touch of javascript for the hamburger on mobile responsiveness

Hosted with vercel

I used Gemini during development as a programming buddy. It helped me bounce design ideas around, figure out the exact syntax needed to URL-encode the blank spaces for my pre-filled WhatsApp link templates, and quickly troubleshoot minor CSS padding bugs. The actual code organization, asset design, copy, and deployment strategy were entirely executed by me.
