# Reina Azul — Personal Brand Portfolio & Community Hub

Hey! This is the repository for my personal portfolio website, built from scratch and deployed live at **https://Reina-azul.vercel.app**. 

I designed this site to serve as a clean, highly visual home base for my creative work, digital content, and performance arts. It also acts as a direct bridge to my community (the Blue Family) by letting people instantly connect with me through custom messaging workflows.

---

## 🚀 Features

*   **Custom WhatsApp Workflows:** Instead of using standard, boring contact forms that people hate filling out, I hardcoded direct WhatsApp Click-to-Chat links. They automatically pre-fill custom message templates depending on whether someone is trying to book a collaboration or join my close-friends style status updates.
*   **Built for Mobile First:** Since most of Her community browses on their phones, I focused heavily on responsive CSS with the aid of gemini . The portfolio pictures stack beautifully on small screens, and the navigation system stays tucked away until it's needed.
*   **Branding & Asset Tuning:** Cleaned up the header completely by replacing old text placeholders with a high-visibility, responsive interlocking logo asset and configured a persistent favicon system across all pages.

---

## 🛠️ Languages used and deployment route

Just clean, lightweight, vanilla frontend code. No bulky frameworks required:
- **HTML:** Semantic architecture across multiple pages (`index.html`, `community.html`, `contact.html`).
- **CSS:** Flexbox modules, custom design variables, and media query breakpoints for cross-device fluidity.
- **Vercel:** Connected directly to this GitHub repo for automated CI/CD deployment pipelines.
