# Reina Azul Portfolio

A responsive multi-page personal portfolio website built to showcase creative work, digital content curation, and community building initiatives for Reina Azul.

## 🔗 Live Demo
- **URL:** https://Reina-azul.vercel.app
- **Platform:** Deployed and hosted on Vercel.

## 🚀 Features
- **Responsive Architecture:** Fully optimized for mobile and desktop screens, featuring clean vertical stacking layouts for visual content.
- **Unified Navigation System:** Consistent header layout cross-linking the Home, Blue Family (Community), and Contact sections.
- **Direct Message Templates:** Integration with WhatsApp using custom-coded Click-to-Chat URLs that pre-fill interactive entry templates for potential brand collaborations and community updates.
- **Custom Identity Branding:** Implementation of personalized asset icons, including a favicon system and high-visibility header logo components.

## 🛠️ Built With
- **HTML:** Semantic site structuring.
- **CSS:** Flexbox layouts, custom variables, design consistency, and responsive media queries.
- **JavaScript:** Client-side interaction workflows.

## 🤖 AI Use Disclosure
AI assistance (Gemini) was utilized during development exclusively as a pair-programmer to assist with debugging responsive layouts, formatting clean mailto/messaging URL syntax, and optimizing custom CSS scaling properties for cross-device compatibility. All structural design choices, branding concepts, and core project components were manually assembled and configured.
